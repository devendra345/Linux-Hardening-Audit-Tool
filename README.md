# Linux Hardening Audit Tool

## Overview
Linux Hardening Audit Tool is a Python-based security auditing tool that helps assess the security posture of a Linux system.

## Features
- Firewall Status Check
- SSH Configuration Audit
- File Permission Verification
- Running Services Analysis
- Security Report Generation

## Technologies Used
- Python 3
- Linux
- Subprocess Module

## Installation

```bash
git clone https://github.com/USERNAME/Linux-Hardening-Audit-Tool.git
cd Linux-Hardening-Audit-Tool
python3 audit.py
